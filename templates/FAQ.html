Cyclic Rotations

A cyclic rotation is a type of operation which can be performed on a sequence of elements. It causes each element in the sequence to move by a fixed amount, in a circular way. This means that when an element reaches the end of the sequence, it wraps around to the beginning. For example, if you cyclically rotate the list [A, B, C, D] to the right by one position, it becomes [D, A, B, C]. Rotating it to the left by one would result in [B, C, D, A].

In the context of drum exercises, the cyclic rotations of an exercise could be thought of as shifting the starting point of a pattern while keeping the overall sequence the same. For example, if you have a sticking pattern like R L L R, a cyclic rotation would involve starting from a different point in the pattern, such as L L R R, L R R L, or R R L L, depending on the direction and number of steps rotated. This helps drummers develop fluidity, ambidexterity, and familiarity with how patterns feel starting on different hands/limbs or beats, which is essential for improvisation and versatility.

What This Tool (Website) Does

This tool allows you to create a drum exercise (using the Editor), then quickly view all cyclic rotations of that exercise (by clicking the "Cyclic Rotate!" button).

The subdivision box allows you to choose a subdivision for the grid (i.e.- the entire grid of the editor is split into even sections according to the current chosen subdivision value). Changing the subdivision automatically deletes all notes currently placed on the grid, so be careful.

Places\beats which you will leave blank will be understood by the editor as rests.

Question 2:

Why should I make an account? What will that give me?

Answer:

Making an account will enable you to save exercises which you create and access them at a later time, when you log in to the site.

Question 3:

Do I have to make an account in order to use the website?

Answer:

You absolutely do not have to make an account in order to use the site. You can still create exercises and view their cyclic rotations, you just won't have the option to save them.
